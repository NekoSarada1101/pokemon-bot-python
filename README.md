## 概要
ランダムなポケモンの情報と画像をSlackで表示する。

## 開発環境
* Python 3.8
* [PokeAPI](https://pokeapi.co/)

## 使用方法
```
git clone https://github.com/NekoSarada1101/random-pokemon-slack-bot.git
cd random-pokemon-slack-bot
python main.py
```
